# 🎓 南亞技術學院 - 自動排課系統 部署說明書

本資料夾已打包所有必要的運行組件。依照以下步驟，您可以在任何安裝了 Docker 的伺服器上快速上線系統。

---

## 🛠️ 準備工作
1.  **安裝 Docker**: 確保您的伺服器已安裝 Docker 與 Docker Compose。
2.  **上傳檔案**: 將整個 `deploy_package` 資料夾上傳至伺服器。

---

## 🚀 部署步驟

### 1. 修改伺服器 IP (重要)
打開 `frontend/index.html`，將檔案中所有的 `http://127.0.0.1:9000` 替換為您 **伺服器的公開 IP**。
*(如果不修改，瀏覽器會嘗試連向您自己的電腦而非伺服器)*

### 2. 啟動系統
在 `deploy_package` 資料夾內執行：
```bash
docker-compose up -d --build
```
這會自動：
*   啟動 PostgreSQL 資料庫。
*   編譯並啟動 Python FastAPI 後端。
*   啟動 Nginx 伺服器並掛載前端網頁。

### 3. 初始化資料庫
系統啟動後，執行以下指令匯入初始的老師、班級與排課規則：
```bash
docker exec -it course_backend python mock_data.py
```

---

## 🌐 存取網址
*   **前端網頁**: `http://您的伺服器IP/`
*   **API 文件**: `http://您的伺服器IP:9000/docs`

---

## 📂 目錄結構說明
*   `backend/`: 包含所有 Python 排課邏輯與 API 程式碼。
*   `frontend/`: 包含網頁前端檔案。
*   `docker-compose.yml`: 定義三層架構（DB + API + Web）的整合設定。
*   `Dockerfile`: 定義後端運行的標準化環境。
